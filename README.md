# TinyURL Clone

### Getting Started
* Uses Ruby 3.0.1 & Rails 6.1.4.1
* Fork the repo from Github
* cd into `tinyurl`
* `bundle install`
* `yarn install`
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`
* `rails server`
